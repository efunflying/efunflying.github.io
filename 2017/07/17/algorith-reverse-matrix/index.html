一道矩阵转置的算法题

今天和新来的同事闲聊中听到一道算法题，讲一个2*5的矩阵，怎样原地转置。画了一下问题之后，思路就打开了。

看起来有这样的关系：

假设原位置是k，则
if (k == 2i)
{
tartget[k] = i;
}
if (k == 2i + 1)
{
target[k] = 5+i;
}

注意到每个轮转最后都是一个闭环(应该是可以证明的):
以i=0为例，2*0+1=1, target(1)=5; target(5)=7; target(7)=8; target(8)=4; target(4)=2; target(2)=1;

因此有：
count = 10
count = count - 2 /首尾两单元不用动/
for(i=0; 2i+1<5 &&="" count="">0 ; i++)
{
transfer(2i+1);
}

其中
transfer(k)
{
for(i = k; k!=target(i); i=target(i))
{
tmp = array[target(i)];
array[target(i)] = tmp;
count –; /某个元素已经就位/
}
array[k] = tmp;
count –; /某个元素已经就位/
}
